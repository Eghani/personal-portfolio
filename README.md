## ✨ Features

### 🎨 Design & User Experience

- **Responsive Design** - Seamlessly adapts to all screen sizes (mobile, tablet, desktop)
- **Modern Dark Theme** - Eye-friendly dark mode with blue and purple accent colors
- **Smooth Animations** - Engaging transitions and hover effects throughout
- **Icon-Based Navigation** - Intuitive navigation with icon tooltips on desktop
- **Mobile Menu** - Smooth slide-in mobile navigation with staggered animations
- **Sticky Navigation** - Always-accessible navbar on mobile and desktop

### 📱 Mobile Optimizations

- **Enhanced Mobile Menu** - Full-screen menu with smooth animations
- **Touch-Optimized** - Large touch targets for better mobile UX
- **Overlay Background** - Semi-transparent overlay when menu is active
- **Body Scroll Lock** - Prevents background scrolling when menu is open
- **Escape Key Support** - Close menu with ESC key for better accessibility

### 🎯 Portfolio Sections

1. **Hero Section** - Animated introduction with social links
2. **About** - Professional summary with key statistics (Projects, Experience, Skills)
3. **Journey Timeline** - Career progression with current position highlighting
4. **Education** - Academic background and achievements
5. **Skills** - Interactive skill icons with progress bars and categories
6. **Achievements** - Notable accomplishments and certifications
7. **Projects** - Showcase of work with live demos and GitHub links
8. **Contact** - Multiple ways to get in touch with social media integration

### 🎭 Interactive Elements

- **Hover Effects** - Smooth transitions on buttons, cards, and icons
- **Progress Bars** - Animated skill proficiency indicators
- **Timeline Markers** - Visual journey representation with date markers
- **Project Cards** - Hover animations with gradient borders
- **Social Links** - Animated social media buttons
- **CTA Section** - Prominent call-to-action for collaboration

### ♿ Accessibility

- **Semantic HTML** - Proper heading structure and semantic elements
- **ARIA Labels** - Screen reader friendly navigation
- **Keyboard Navigation** - Full keyboard support for all interactive elements
- **Alt Attributes** - Descriptive text for all images
- **Focus States** - Clear focus indicators for keyboard users

## 🔧 Setup & Installation

1. **Clone the repository**

   ```bash
   git clone https://github.com/Eghani/personal-portfolio.git
   ```

2. **Navigate to the project**

   ```bash
   cd personal-portfolio
   ```

3. **Open in browser**

   ```bash
   # Simply open index.html in your browser
   # Or use a local server
   python -m http.server 8000
   # Or
   npx serve
   ```

4. **Customize**
   - Update `index.html` with your information
   - Modify colors in `style.css`
   - Replace images in `assets/images/`
